Detailed Description

This layer is situated over the physical layer and acts.

as a serial datalink layer

This layer frames the information and operates over the hardware buffer streams of serial information. The framing is conducted using the SLIP protocol.

Message structure:

N bytes of information (stuffed accordint to the protocol) END byte
N bytes 8 bit
